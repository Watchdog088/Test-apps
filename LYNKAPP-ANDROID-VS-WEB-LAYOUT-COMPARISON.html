Android Architecture

⚙️ Native Layer (Capacitor 5)

  • Capacitor WebView wraps the React SPA
  • MainActivity.java already scaffolded
  • google-services.json copied to android/app/
  • AndroidManifest.xml — all permissions added
  • @capacitor/camera, status-bar, splash-screen — need npm install
  • Back button hardware exits app (no listener yet)
  • targetSdkVersion must be 34 for Play Store

Status BarOverlaps TopNav
Safe Area BottomNeeds adjustResize
Keyboard PushBehind keyboard
Back ButtonExits app
Touch Targets (44px)✓ Set in CSS
Scrollbars Hidden✓ global.css

🔧 Android-Specific Fixes Needed

  • Install: npm i @capacitor/camera @capacitor/push-notifications @capacitor/status-bar @capacitor/splash-screen @capacitor/keyboard
  • Run: npx cap sync android
  • Add App.addListener('backButton', handler) in main.jsx
  • StatusBar.setBackgroundColor('#0a0818') on app init
  • Add windowSoftInputMode="adjustResize" to AndroidManifest
  • App icon placeholder — replace mipmap-*/ic_launcher.png

🚀 Build Commands

  • Debug APK: cd android && ./gradlew assembleDebug
  • Release AAB: ./gradlew bundleRelease
  • Quick build: run android-build.bat
  • Play Store: Upload AAB to Internal Testing track
