## 🎨 ICON STYLE GUIDE

| Rule | Value |
|------|-------|
| Icon type | Emoji (Unicode) — no SVG icon library dependency |
| Sidebar icon size | 20px font-size |
| ☰ More icon size | 17px font-size |
| Minimum touch target | 44 × 44px (Apple HIG standard) |
| Active icon color | `#818cf8` (indigo-400) |
| Inactive icon color | Opacity 0.55 |
| Live/danger color | `#f87171` (red-400) |
| Badge font-size | 8–9px, font-weight 800 |

## 📝 NOTES FOR DEVELOPERS

1. **All icons are plain Unicode emoji** — no icon library (Heroicons, FontAwesome, etc.) is needed
2. **To add a new icon:** Simply use the emoji character directly in JSX e.g. `<span>🔔</span>`
3. **Consistent sizing:** Use `fontSize: '20px'` for nav icons, `fontSize: '17px'` for small UI icons
4. **Accessibility:** Always include `aria-label` on icon-only buttons e.g. `aria-label="Notifications"`
5. **Touch targets:** Wrap icons in buttons with `minHeight: '44px'` and `minWidth: '44px'`
6. **Live pulse animation:** CSS class `livePulse` in global.css — use for any real-time status indicator
