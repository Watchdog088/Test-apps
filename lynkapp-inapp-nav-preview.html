📌 Top Nav — Header Buttons

  • Unix Icon Btns — 38×38px, rounded corners, translucent purple tint. Used for: Compose ✏️, Search 🔍, Notifications 🔔, Feedback 💬
  • Feedback btn — same unix-btn shape but green border tint to distinguish beta feature
  • Avatar circle — 40×40px circle, purple ring, shows first initial or photo. Navigates to Profile
  • LynkApp Logo — gradient text wordmark (not a button, but the brand anchor)

📌 Bottom Nav — Navigation Tabs

  • Active tab — purple text + top gradient underline bar (Feed, Search, Alerts, Me)
  • Inactive tab — 30% white opacity, dims when not selected
  • Center FAB ➕ — raised 48px gradient circle, lifts above nav, rotates 90° on hover → compose/post
  • Notification dot — pink 7px circle on Alerts tab when unread

📌 In-Feed Buttons

  • + Follow pill — small 11px gradient purple pill, appears beside users you don't follow
  • Post actions row — ghost icon buttons (❤ 💬 ↗ 🔖) with hover purple tint, like turns pink
  • Filter chips — pill tags: active = purple tint, inactive = faint white border
