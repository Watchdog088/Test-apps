## STEP 9: Test Your License Key ⏱️ 5 minutes

### **Quick Test to Verify Key Works**

**Create Test HTML File:**

1. Create new file: `test-deepar.html`
2. Paste this code:

```html
<!DOCTYPE html>
<html>
<head>
    <title>DeepAR Test</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            text-align: center;
            padding: 50px;
        }
        #deepar-canvas {
            width: 640px;
            height: 480px;
            border: 2px solid #000;
            margin: 20px auto;
        }
        .controls {
            margin: 20px;
        }
        button {
            padding: 10px 20px;
            margin: 5px;
            font-size: 16px;
            cursor: pointer;
        }
        #status {
            margin: 20px;
            font-size: 18px;
        }
    </style>
</head>
<body>
    <h1>🥽 DeepAR Test</h1>
    <div id="status">Initializing...</div>
    
    <canvas id="deepar-canvas"></canvas>
    
    <div class="controls">
        <button onclick="switchEffect('aviators')">😎 Aviators</button>
        <button onclick="switchEffect('flower_face')">🌸 Flowers</button>
        <button onclick="switchEffect('lion')">🦁 Lion</button>
        <button onclick="takePhoto()">📸 Take Photo</button>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/deepar@5/js/deepar.js"></script>
    <script>
        let deepAR;

        // REPLACE WITH YOUR LICENSE KEY!
        const licenseKey = 'YOUR_LICENSE_KEY_HERE';

        // Initialize DeepAR
        (async function() {
            try {
                deepAR = await window.deepar.initialize({
                    licenseKey: licenseKey,
                    canvas: document.getElementById('deepar-canvas'),
                    effect: 'https://cdn.jsdelivr.net/npm/deepar@5/effects/aviators'
                });

                document.getElementById('status').innerHTML = 
                    '✅ <b>DeepAR Connected!</b> Your license key works!';
                
                console.log('✅ DeepAR initialized successfully!');
            } catch (error) {
                document.getElementById('status').innerHTML = 
                    '❌ <b>Error:</b> ' + error.message;
                
                console.error('❌ DeepAR initialization failed:', error);
            }
        })();

        // Switch AR effects
        async function switchEffect(effect) {
            const effects = {
                aviators: 'https://cdn.jsdelivr.net/npm/deepar@5/effects/aviators',
                flower_face: 'https://cdn.jsdelivr.net/npm/deepar@5/effects/flower_face',
                lion: 'https://cdn.jsdelivr.net/npm/deepar@5/effects/lion'
            };

            await deepAR.switchEffect(effects[effect]);
            console.log('Switched to:', effect);
        }

        // Take photo
        async function takePhoto() {
            const photo = await deepAR.takeScreenshot();
            console.log('Photo taken!', photo);
            
            // Download photo
            const a = document.createElement('a');
            a.href = photo;
            a.download = 'deepar-photo.png';
            a.click();
        }
    </script>
</body>
</html>
```

3. **IMPORTANT:** Replace `YOUR_LICENSE_KEY_HERE` with your actual key!

**If You See Error:**

## 🐛 TROUBLESHOOTING:

### **Common Issues:**

**Issue 1: "Invalid License Key"**
```
Solution:
- Check key is copied correctly (no spaces)
- Verify project platform matches (Web, iOS, etc.)
- Try regenerating key in dashboard
```

**Issue 2: "Camera Not Working"**
```
Solution:
- Allow camera permissions in browser
- Use HTTPS (or localhost for testing)
- Try different browser (Chrome recommended)
- Check camera is not used by another app
```

**Issue 3: "Effects Not Loading"**
```
Solution:
- Check internet connection
- Verify effect URL is correct
- Try using CDN effects first
- Check browser console for errors
```

**Issue 4: "Exceeded Monthly Limit"**
```
Solution:
- You've reached 10 active users
- Upgrade to Starter tier ($99/month)
- Or wait until next month
- Check dashboard for exact count
```
